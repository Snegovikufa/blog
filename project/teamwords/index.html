TeamWords

Rustam Safin Feb 26, 2015
Github Code Repository Website Website Link

TeamWords is a free cross-platform desktop Slack client with native notifications.

TeamWords was created when Slack had not released official native client. Web-version has only short time notifications (< 10 seconds). If you miss notification, you will not notice that someone had written you a message.

Technology stack used

  • C++
  • Qt5 (Widgets + Webkit)
